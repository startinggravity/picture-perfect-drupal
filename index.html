An Easy Fix


            img { max-width: 100%; }

Except...

This won't work on IE 6 or 7

This little trick makes images scalable! The image's size is determined by the container around it. Setting max-width to 100% ensures the image does not scale larger than its actual size [reveal text] Won't work on IE 6 or 7.

An Easy Fix for IE 6 & 7


            img { width: 100%; }
Use width 100%.
